HabbGames

Full-stack Just for fun Self-made Database/SQL

HabbGames was a fun little website I created when I was 15 years old, it was the first ever "full-stack" program I made - without knowing what that meant at the time. The application consisted of a user-system with login/logout functionality, had a forum page, article-pages, and many more functions. It was used as a Habbo-co-page - Habbo is/was a popular online RPG-game. I don't have a lot of code or anything from the time of creating the website, but I have a few pictures and lots of memories. I decided to include this as a project of mine, as it is my first ever working application - I'm both proud and embarrassed when talking about HabbGames.
Oh, and it was created using HTML, CSS, JS, and lots of PHP, alongside a functional database.
